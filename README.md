# SUPAR MAYRO - Ultimate Platformer

A 2D side-scrolling platformer with RIDICULOUS traps and hazards! Boss battles, weapons, double jump, and more!

## Features

- **10 Levels** with increasing difficulty
- **Procedural Generation** - Each playthrough creates a unique level layout
- **Boss Battles** - Face bosses at levels 3, 6, and 9!
- **Human Player** - Custom character with skin, hair, shirt, and pants
- **Double Jump** - Jump again in mid-air!
- **Weapon System** - Shoot with Z/X keys to defeat enemies from afar
- **6+ Enemy Types** - Goombas, Angry Goombas, Crazy Koopas, and more!
- **Ground Detection** - Mobs only walk on the ground
- **Collectibles** - Coins, BANANA PEELS, MYSTICAL FRUITS!
- **Flying Hazards** - BATS that swoop down!
- **Moving Platforms** - Blue platforms that move horizontally!
- **TRAPS** - Spike strips, deadly orbs, X-shaped traps!
- **FALLING SPIKES** - Spikes that fall from the sky!
- **Spikes** - Floor spikes AND ceiling spikes!
- **Synthesized Sound Effects** - Music, jump, coin, shoot, hit, death, win sounds!
- **No External Assets** - Everything generated programmatically

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys / A,D | Move left/right |
| Space / W | Jump |
| Double Tap Space/W | Double Jump |
| Z / X | Shoot weapon |
| P | Pause game |
| R | Restart (after game over/win) |

## Objective

- Collect coins (+50-100 points)
- Collect fruits (+150 points)
- Defeat enemies by jumping on them (+200-250 points)
- Shoot enemies with your weapon
- Avoid ALL the traps!
- Defeat bosses at levels 3, 6, and 9!
- Reach the flag at the end (+1000 points)
- Complete all 10 levels to win!
